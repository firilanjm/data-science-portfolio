# Personal Expense Tracker (Java GUI)

## Overview
A desktop-based personal expense tracker built using Java Swing that allows users to record, view, filter, and manage expenses through an interactive graphical user interface.

## Features
- Add and remove expenses
- Filter expenses by category and date range
- View all expenses in a table
- Event logging for user interactions

## Technical Design
- Object-oriented architecture
- Event-driven GUI using Java Swing
- Table-based visualization using JTable and DefaultTableModel

## Why This Project Matters
This project demonstrates software engineering fundamentals and the ability to build user-facing applications that collect and manage data, complementing data science projects focused on analysis and modeling.

## Tools
Java, Java Swing
